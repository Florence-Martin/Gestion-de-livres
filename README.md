# GestionDeLivres 

Ce dépôt contient le code source de l'application BooksApp. Il est divisé en deux parties :
- **BooksBackend** : Projet ASP.NET Core gérant la logique backend.
- **BooksFrontend** : Projet React/TypeScript pour l'interface utilisateur.

## Installation

Pour exécuter localement l'application, suivez ces étapes :
1. Clonez ce dépôt.
2. Installez les dépendances pour le frontend et le backend.
3. Lancez l'application.

## Technologies utilisées

- ASP.NET Core
- React
- TypeScript
